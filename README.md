# 🔐 Password Manager

> ⚠️ **Це гілка активної розробки (`develop`).** Деякі функції можуть бути нестабільними або ще не реалізованими. Стабільну версію дивіться в гілці `main`.

Локальний менеджер паролів для Windows, написаний на **C# / WPF**. Дозволяє зберігати логіни та паролі для різних сервісів у зашифрованому вигляді на вашому комп'ютері та швидко копіювати їх у буфер обміну.

## 📋 Опис

Застосунок створений як простий і зрозумілий інструмент для зберігання облікових даних без використання хмарних сервісів. Усі дані залишаються локально на вашому ПК у зашифрованому файлі.

## ✨ Можливості

- 🚧 Додавання нових записів (в розробці)
- 🚧 Редагування записів (в розробці)
- 🚧 Видалення записів (в розробці)
- 🚧 Показати / сховати пароль (в розробці)
- 🚧 Копіювання логіна та пароля (в розробці)
- 🚧 Пошук записів (в розробці)
- 🚧 Зберігання даних у зашифрованому файлі (в розробці)
