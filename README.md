# PassKeeper# 🔐 Password Manager

Локальний менеджер паролів для Windows, написаний на **C# / WPF**. Дозволяє зберігати логіни та паролі для різних сервісів у зашифрованому вигляді на вашому комп'ютері та швидко копіювати їх у буфер обміну.

## 📋 Опис

Застосунок створений як простий і зрозумілий інструмент для зберігання облікових даних без використання хмарних сервісів. Усі дані залишаються локально на вашому ПК у зашифрованому файлі.

## ✨ Можливості

- ➕ Додавання нових записів (сервіс, логін, пароль, примітка)
- ✏️ Редагування існуючих записів
- 🗑️ Видалення записів (з підтвердженням)
- 👁️ Показати / сховати пароль
- 📋 Копіювання логіна та пароля в буфер обміну одним кліком
- 🔍 Пошук записів за назвою сервісу або логіном у реальному часі
- 🔒 Зберігання даних у зашифрованому (AES) локальному файлі

## 🛠️ Технологічний стек

| Компонент | Технологія |
|---|---|
| Мова | C# (.NET 8) |
| UI | WPF |
| Архітектура | MVVM |
| Зберігання даних | Локальний файл (JSON, зашифрований AES) |

## 🚀 Встановлення та запуск

### Вимоги
- Windows 10/11
- [.NET 8 SDK](https://dotnet.microsoft.com/download) або новіша версія

### Кроки запуску

```bash
# Клонувати репозиторій
git clone https://github.com/yourusername/password-manager.git
cd password-manager

# Відновити залежності
dotnet restore

# Зібрати проєкт
dotnet build

# Запустити застосунок
dotnet run
```

## 💾 Зберігання даних

Усі записи зберігаються в одному зашифрованому файлі за шляхом:

```
%AppData%\PasswordManager\vault.dat
```

Дані шифруються алгоритмом **AES** перед записом на диск і автоматично розшифровуються при запуску застосунку.

> ⚠️ Це MVP-версія без майстер-пароля при вході. Файл захищений шифруванням від випадкового перегляду, але не призначений для зберігання критично важливих паролів у середовищах з кількома користувачами.
